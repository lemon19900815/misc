# jupyter快捷键

- `ctrl+?`：[ 取消 ] 注释代码
- `shit+tab`：显示函数说明信息
- `%run *.py`：jupyter内部执行外部py文件

- 双击运行之后的结果会收起来

## conda

- `conda create -n env_name python=X.X`（2.7、3.6等)命令创建python版本为X.X

- `conda activate env_name`：进入虚拟环境
- `conda deactivate`：退出当前虚拟环境
- `conda env list`：查看有哪些虚拟环境

- `conda list`：查看安装了哪些包
